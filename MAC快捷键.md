# MAC快捷键

#### 系统

| 功能       | 快捷键                      |
| :------- | :----------------------- |
| 退出应用     | Command + Q              |
| 窗口最大/最小化 | Control + Command + F    |
| 放大/缩小/还原 | Command + [+ / - / 0]    |
| 最小化      | Command + M              |
| 截图全屏     | Shift + Command + 3      |
| 截图选择区域   | Shift + Command + 4      |
| 清空废纸篓    | Shift + Command + Delete |
| 删除选中文件   | Command + Delete         |
| 向上/下翻页   | Fn + ↑ / ↓               |
| 应用内TAB切换 | Ctrl + Tab               |
| 前往文件夹    | Shift + Command + G      |
| 打开终端     | Command + Space，搜索ter，回车 |
| 终端新建     | Command + T              |
| 终端拆分     | Command + D              |
| 终端取消拆分   | Command + Shift + D      |
| 链接服务器    | Command + K              |
|          |                          |
|          |                          |
|          |                          |
|          |                          |
