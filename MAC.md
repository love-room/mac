# MAC

#### 显示所有隐藏文件：

```shell
defaults write com.apple.finder AppleShowAllFiles TRUE
```

#### 隐藏所有隐藏文件：

```shell
defaults write com.apple.finder AppleShowAllFiles NO
```

#### 生效：

```shell
killall Finder
```

#### 显示所有文件：

```shell
ls -all
```

#### git失效

```shell
xcode-select --install
```
